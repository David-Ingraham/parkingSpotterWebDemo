Real-Time Parking Detection

A live demo showcasing a fine-tuned YOLOv8 model for parking spot detection.

How it Works

  1. Start typing a camera location in the search box below (e.g., "5th Ave", "Broadway").
  2. Select a camera from the autocomplete suggestions that appear.
  3. Click the "Detect Parking" button to see a real-time image with detected spots.
